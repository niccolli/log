# Raspberry PiでSPIディスプレイをフル活用する



## 画面

```
#dtoverlay=vc4-kms-v3d
dtoverlay=vc4-fkms-v3d
```

解像度は640x480に設定

```
cmake -DILI9341=ON -DGPIO_TFT_DATA_CONTROL=25 -DGPIO_TFT_RESET_PIN=24 -DSPI_BUS_CLOCK_DIVISOR=8 -DARMV8A=ON -DUSE_DMA_TRANSFERS=ON ..
```
